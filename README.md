## ğŸ’¡ How to Use

1. **Visit** [https://data-viz-charts.netlify.app/](https://data-viz-charts.netlify.app/)
2. **Upload** your CSV or Excel file using the file upload button
3. **View** your data in the interactive table with search and filter options
4. **Configure** your chart by selecting:
   - Chart type (Bar, Line, Pie, etc.)
   - X-Axis column (categories/labels)
   - Y-Axis column (values)
   - Optional filters to refine your data
5. **Generate** your chart and interact with it
6. **Export** your chart as SVG, PNG, or PDF
